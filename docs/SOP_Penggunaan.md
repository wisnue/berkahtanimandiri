## 👥 PANDUAN UNTUK KETUA

### 1. Login Sistem
1. Buka browser, akses: http://localhost:5173
2. Masukkan username/NIK dan password
3. Klik tombol **Masuk**

### 2. Dashboard
- Lihat ringkasan:
  - Total anggota
  - Total luas lahan KHDPK
  - Status PNBP tahun berjalan
  - Saldo kas KTH
  - Aktivitas terbaru

### 3. Kelola Anggota
1. Klik menu **Anggota**
2. **Tambah Anggota Baru**:
   - Klik tombol **+ Tambah Anggota**
   - Isi form lengkap (NIK, nama, alamat, dll)
   - Upload foto KTP
   - Klik **Simpan**
3. **Edit Data Anggota**:
   - Klik nama anggota
   - Ubah data yang perlu diubah
   - Klik **Update**
4. **Cetak Kartu Anggota**:
   - Klik detail anggota
   - Klik **Cetak Kartu**
   - Kartu akan ter-generate dengan QR Code

### 4. Kelola Lahan KHDPK
1. Klik menu **Lahan KHDPK**
2. **Tambah Lahan**:
   - Klik **+ Tambah Lahan**
   - Pilih anggota pemilik
   - Isi data lahan (kode, luas, petak, dll)
   - Upload SK KHDPK & peta lahan
   - Klik **Simpan**

### 5. Kelola PNBP
1. Klik menu **PNBP**
2. **Generate PNBP Tahunan**:
   - Klik **Generate PNBP**
   - Pilih tahun
   - Sistem otomatis hitung berdasarkan luas lahan
3. **Input Pembayaran**:
   - Klik data PNBP
   - Upload bukti setor
   - Isi tanggal bayar
   - Klik **Verifikasi Pembayaran**

### 6. Kelola Keuangan
1. Klik menu **Keuangan**
2. **Catat Kas Masuk**:
   - Klik **+ Transaksi**
   - Pilih jenis: Kas Masuk
   - Isi jumlah, kategori, sumber dana
   - Upload bukti
   - Klik **Simpan**
3. **Catat Kas Keluar**:
   - Sama seperti kas masuk, pilih Kas Keluar
4. **Cetak Laporan**:
   - Pilih periode
   - Klik **Export PDF** atau **Export Excel**

### 7. Kelola Dokumen
1. Klik menu **Dokumen**
2. **Upload Dokumen**:
   - Klik **+ Upload Dokumen**
   - Pilih jenis (SK, AD/ART, Laporan, dll)
   - Upload file
   - Isi metadata
   - Klik **Simpan**

### 8. Logout
- Klik nama Anda di sidebar
- Klik **Keluar**

## 📝 PANDUAN UNTUK SEKRETARIS

### Tugas Utama:
1. **Administrasi Anggota**
2. **Pencatatan Kegiatan**
3. **Manajemen Dokumen**

### Workflow:
1. **Update Data Anggota**:
   - Terima formulir anggota baru
   - Input ke sistem
   - Cetak kartu anggota
   - Arsipkan dokumen

2. **Catat Kegiatan**:
   - Klik menu **Kegiatan**
   - Klik **+ Tambah Kegiatan**
   - Isi detail kegiatan (tanam/panen/rapat)
   - Upload foto dokumentasi
   - Klik **Simpan**

3. **Upload Notulen Rapat**:
   - Klik menu **Dokumen**
   - Upload file notulen
   - Tandai sebagai "Notulen Rapat"
